# Self Portfolio

A portfolio website for my own.

## Environment Variables

Set these for Giscus comments:

```env
PUBLIC_GISCUS_REPO=your-username/your-repo
PUBLIC_GISCUS_REPO_ID=your-repo-id
PUBLIC_GISCUS_CATEGORY_ID=your-category-id
```
